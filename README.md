# Next.js 게시판

### Next.js를 이용한 게시판 프로젝트입니다. DB는 'mongoDB'를 사용했고 API 라우팅 기능을 통해 데이터를 안전하게 저장,관리합니다. Next-auth를 이용한 회원 인증 기능으로 보안성을 강화했고 Next.js의 서버사이드 렌더링(SSR) 기능을 활용해 SEO 최적화했습니다.
